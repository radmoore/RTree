NewickTokenizer

NewickTokenizer - Parse a newick formatted string into its tokens.

This code is inspired by the newick-ruby gem written by Jonathan Badger (github.com/jhbadger/Newick-ruby), and is based on the newick format described here evolution.genetics.washington.edu/phylip/newick_doc.html
